## Feature

- basic commands/tools, such as vim/tmux/git/make/automake/tcpdump/ack-grep/upx/ntpdate
- vim plugins, such as vundle/YouCompleteMe/vim-go
- golang source build
- nice login banner

## Usage

```bash
git clone https://github.com/polym/sysinit.git
./start.sh
```
